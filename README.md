# 🎟️ ElitePass - Premium Event Booking UI

Welcome to **ElitePass**, a high-performance, interactive event booking system designed for a premium user experience. This project was developed as part of my internship at **Sysslan IT Solutions**.

## ✨ Key Features
- **Smooth Animations:** Integrated scroll reveal effects that make event cards pop up as you browse.
- **Dynamic Seat Selection:** An interactive seat map where users can select multiple seats with real-time price updates.
- **Tiered Pricing:** Automatic calculation for Premium (₹1199+) and Economy (₹999) seating.
- **Secure Payment Simulation:** Integrated UPI QR code system for a seamless checkout flow.
- **Interactive Feedback:** Celebratory confetti animations triggered upon successful booking and final confirmation.
- **Modern UI:** Built with a dark-themed, glassmorphism-inspired design using Google Fonts (Poppins).

## 🛠️ Tech Stack
- **HTML5:** Semantic structure for better accessibility.
- **CSS3:** Custom animations, flexbox, and grid layout for a responsive experience.
- **JavaScript (Vanilla):** DOM manipulation, event listeners, and dynamic UI logic.
- **Canvas-Confetti API:** For the high-quality celebration effects.

## 🚀 How to Run
1. Clone this repository or download the ZIP.
2. Open `index.html` in any modern web browser.
3. Scroll down to see the event "Pop-up" effect.
4. Click "Select Seats" to experience the booking flow.
